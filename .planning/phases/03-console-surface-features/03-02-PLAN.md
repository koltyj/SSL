```python
# Keymap name constants (from Java source):
# Valid names: "keymap1", "keymap2", "keymap3", "keymap4"
# "NONE" = no keymap configured (cannot be used to open edit session)

class MessageCode(IntEnum):
    SEND_GET_EDIT_KEYMAP_NAME = 600
    ACK_GET_EDIT_KEYMAP_NAME = 601
```

Key type values:
- 0 = blank (unassigned)
- 1 = MIDI function (indexed from MIDI function list)
- 2 = USB HID keycode string
- 3 = submenu
