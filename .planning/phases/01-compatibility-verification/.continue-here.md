<decisions_made>

- CalDigit TS4 2.5GbE Ethernet CANNOT link with Matrix 100Mbps — speed negotiation failure. Anker USB Ethernet adapter is the working path.
- Anker adapter enumerates as "USB 10/100/1G/2.5G LAN" service, interface en11 (but interface number may change on replug — service name is stable)
- Gateway set to 0.0.0.0 on the Ethernet service to prevent macOS routing internet traffic through en11 and killing Wi-Fi
- Ableton uses MCU on Port 3 (MackieControl) + Port 4 (MackieControlXT) — MCU hard limit is 8 faders per device
- Pro Tools HUI initially configured on Ports 1+2 — these may be wrong. Need MIDI Monitor to identify actual HUI ports.
- ipMIDI multicast appears to work on en0 (Wi-Fi) even when Anker is disconnected — network topology is unclear but functional. MatrixRemote requires the direct Ethernet path (TCP to 192.168.1.2).
- Anker adapter is flaky on replug — sometimes doesn't re-enumerate. Plugging directly into Mac USB-C port (bypassing dock) helps.
- delta-ctrl multicast route (225.0.0.38) was already persistent from Plan 01-01 — route is NOT the issue for delta-ctrl
</decisions_made>
