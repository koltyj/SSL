2. **ssl-matrix-client/client.py**: Add three callback hooks to `SSLMatrixClient.__init__`:
   ```python
   self._on_state_changed: "Callable | None" = None
   self._on_desk_offline: "Callable | None" = None
   self._on_desk_online: "Callable | None" = None
   ```
   In `_recv_loop`, after the `with self._lock:` dispatch block (line ~237, outside the lock), fire the state-changed hook:
   ```python
   if self._on_state_changed:
       self._on_state_changed()
   ```
   This MUST be outside the lock — `post_message` enqueues but the handler on the other side will acquire the lock to read state. Posting inside the lock would deadlock.

   In `_trigger_reconnect`, after setting `state.desk.online = False` and releasing the lock, fire:
   ```python
   if self._on_desk_offline:
       self._on_desk_offline(self._reconnect_attempts)
   ```

   In `_on_desk_came_online`, after clearing reconnect flags, fire:
   ```python
   if self._on_desk_online:
       self._on_desk_online()
   ```
