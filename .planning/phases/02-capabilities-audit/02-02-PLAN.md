**Tier 4 — High-risk mutations**

**IMPORTANT:** These operations can affect active projects and require a console restart. Do NOT test while DAWs are recording. Ensure current project state is saved.

**Automation mode change + console restart:**
```
ssl> automode                         # record current mode
ssl> automode delta                   # (or legacy — switch to opposite)
# Note: mode does NOT take effect until restart
ssl> disconnect
```
Then restart the console:
```
ssl> connect
ssl> restart                          # SEND_RESTART_CONSOLE — confirm with 'y'
ssl> disconnect
```
Wait ~30 seconds for console reboot. Then reconnect:
```
ssl> connect
ssl> automode                         # verify mode changed
```
Record PASS/FAIL for SEND_SET_AUTOMATION_MODE and SEND_RESTART_CONSOLE.

Restore original mode:
```
ssl> automode <original>
ssl> restart                          # confirm 'y', disconnect, wait, reconnect
ssl> connect
ssl> automode                         # verify restored
```

**Project CRUD (test on disposable project only):**
```
ssl> projects                         # baseline — record existing projects
ssl> new_project AUDIT-TEMP           # ACK_MAKE_NEW_PROJECT_WITH_NAME
ssl> projects                         # verify AUDIT-TEMP exists
ssl> new_title AUDIT-TEMP TESTTITLE   # ACK_MAKE_NEW_PROJECT_TITLE_WITH_NAME
ssl> select_title AUDIT-TEMP TESTTITLE  # ACK_SELECT_PROJECT_TITLE
ssl> project_info                     # verify current project = AUDIT-TEMP / TESTTITLE
ssl> delete_title AUDIT-TEMP TESTTITLE  # ACK_DELETE_PROJECT_TITLE — confirm 'y'
ssl> delete_project AUDIT-TEMP        # ACK_DELETE_PROJECT — confirm 'y'
ssl> projects                         # verify AUDIT-TEMP is gone
```
Record PASS/FAIL for each project ACK handler.

For remaining project ACKs that need different entry points (MAKE_NEW_PROJECT, MAKE_NEW_PROJECT_TITLE, COPY_PROJECT_TITLE, MAKE_NEW_PROJECT_WITH_PRESET_OPTS): test if the CLI supports them, or use raw commands. If not testable without risk, mark as SKIP with reason.

**Re-select original project after testing:**

```
ssl> project_info                     # confirm restored
```
